Pipes in Angular

Angular Pipes are a powerful feature that allows you to transform and format data in your Angular templates
UpperCase Pipe:

{{ 'Hello' | uppercase }}

LowerCase Pipe:

{{ 'Hii' | lowercase }}

Slice Pipe:

{{ 'AML360z' | uppercase | slice:0:5 }}

Decimal Pipe:

{{ 3.14159 | number:'1.2-2' }}

Percentage Pipe:

{{ 0.75 | percent }}

Json Pipe:

{{ { name: 'John', age: 30 } | json }}

Currency Pipe:

{{ 100 | currency:'USD':'symbol' }}

{{ 100 | currency:'INR':'symbol':'1.2-2' }}

Date Pipe:

{{ today | date:'fullDate' }}

{{ today | date:'shortTime' }}

{{today|date:'dd-yyyy-MM'}}

Filter Pipe
    List of Items
  • {{ item }}
